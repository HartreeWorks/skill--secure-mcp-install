# Secure MCP Install

A Claude Code skill for securely installing MCP servers from third-party sources with automated security scanning and commit pinning.

## Documentation

See [SKILL.md](./SKILL.md) for complete documentation and usage instructions.

## Installation

To use this skill with Claude Code, clone it into your skills directory:

```bash
git clone https://github.com/HartreeWorks/claude-skill--secure-mcp-install.git ~/.claude/skills/secure-mcp-install
```

Then restart Claude Code to load the skill.
